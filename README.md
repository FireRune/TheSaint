# The Saint

A mod for the game "The Binding of Isaac: Rebirth" that includes new characters and items.

**This mod is currently WIP!**
The final amount of content will change in the future.

This version is designed to be used with the Repentance and Repentance+ DLCs.

Using [IsaacScript](https://isaacscript.github.io) (version 87.8.0)

## Current list of features

- 2 new characters: "The Saint" and "Tainted Saint"
- 6 new collectibles
- 1 new trinket
- 2 new cards/runes
- integration for [External Item Descriptions](https://github.com/wofsauge/External-Item-Descriptions)

For a list of planned features, see [here](https://github.com/FireRune/TheSaint/blob/main/TODO.md)
